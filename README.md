# 💰 Calculadora de Reventa de Sneakers

**¿A cuánto vendo este par?** Herramienta gratis para revendedores: metes tus costos
y el margen que quieres, y te da el precio de venta justo con desglose visual.

## ✨ Qué hace

- **Precio sugerido** con redondeo comercial (hacia arriba, a miles)
- **Desglose visual** del precio: costo / envío+empaque / comisión / tu ganancia
- **Punto de equilibrio** — el precio por debajo del cual pierdes plata
- **Modo inverso**: "si lo vendo a $X, ¿cuánto gano?" con veredicto honesto
- **Copiar resumen** listo para pegar en WhatsApp
- Recuerda tus últimos números (localStorage) · 100% responsive · cero dependencias

## 🎨 Detalles

Los colores del desglose están validados para daltonismo (deuteranopia/protanopia/
tritanopia) y contraste sobre fondo oscuro. Una sola página HTML, sin frameworks.
